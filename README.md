# Приложение «Работа мечты»

Приложение **«Работа мечты»** — это сервис для взаимодействия кандидатов и кадровиков.  
В системе реализован базовый функционал работы с вакансиями и резюме.

## Основные возможности

### Кандидаты
- публикуют свои резюме
- просматривают вакансии
- откликаются на интересующие вакансии

### Кадровики
- создают и размещают вакансии
- просматривают резюме кандидатов
- приглашают кандидатов на открытые вакансии
